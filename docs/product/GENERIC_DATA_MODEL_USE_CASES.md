# Generic Data Model Use Cases

## Excel Data Dictionary Cleanup

Import or profile spreadsheet-derived fields, infer draft canonical objects, validate IDs and references, and create `PatchProposal`s for review.

## Dataset-To-Model Gap Detection

Profile a CSV/XLSX file, compare columns to modeled `FieldEndpoint`s and `Attribute`s, and create issues or proposals for missing knowledge.

## Field Ownership Tracking

Find active attributes, endpoints, mappings, value lists, and rules without owners or stewards.

## Mapping Review

Trace source endpoints to target endpoints through mappings, value mappings, transformation logic, and validation rules.

## Validation Rule Coverage

Report attributes or fields without validation coverage and create issue drafts for follow-up.

## Impact Before Changing A Field

Run impact analysis before changing a field, attribute, mapping, value list, rule, or owner.

## Evidence-Backed Decisions

Link issues, decisions, evidence, and change requests to affected model objects.

## AI-Generated Patch Proposals

Convert notes, profiles, or review comments into structured `PatchProposal`s for human review.
