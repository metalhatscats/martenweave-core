# Failure Report Template

Use this when a task cannot be safely completed.

```markdown
## Status
Blocked

## Blocker
Exact reason the task cannot proceed.

## Evidence
- Command run:
- Error summary:
- Files inspected:

## What Was Not Changed
- Confirm source files/canonical files/generated files were not mutated, or list intentional partial changes.

## Human Decision Needed
State the minimum decision or credential needed.

## Recommended Next Step
Smallest safe action to unblock.
```
